Hotel of Hotels Database

Customers
Displays a list of all current customers registered in this database. User is able to add a customer entity (similar to registration), with all attributes and other crucial details.
Rooms
Displays all room entities within the database. User can input additional room entries into the database.
Orders
Displays all current orders within the database. User can input additional order entries into the database.
Distributors
Displays data for all distributors of hotel rooms available in our database system.
Ordered Rooms
The Ordered-Rooms page represents the M:M relationship between the Room and Orders entities. The user has the ability to insert and delete data.
